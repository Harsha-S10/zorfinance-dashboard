# ZorFinance (Student Version)

This is a simple frontend dashboard made for assignment submission.

## Tech Used

- HTML
- CSS
- JavaScript
- Chart.js

## Run

1. Open the folder in VS Code.
2. Open `index.html` in browser.

No backend is needed.

## Features

### Dashboard Overview
- Total Balance, Total Income, Total Expense cards
- Balance trend chart (time based)
- Spending by category chart (categorical)

### Transactions
- Table with date, description, category, type, amount
- Search
- Filter by type
- Sort by date/amount/category
- Empty state when no data matches

### Role Based UI
- Viewer: can only view data
- Admin: can add and edit transaction description

### Insights
- Highest spending category
- Monthly comparison (expense increase/decrease)
- Total number of transactions
- Monthly income vs expense chart

### State Management
- Single JS object for app state
- Local storage for transactions, role, and theme

### UI/UX
- Responsive layout
- Light/dark mode toggle
- Clean and basic design
