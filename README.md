# 💰 ZorFinance

## Overview
ZorFinance is a clean and interactive finance dashboard designed to help users track, analyze, and understand their financial activity.
It provides a structured interface to view financial summaries, manage transactions, and gain useful insights from data.
---

### 📊 Dashboard Overview
- Summary cards:
  - Total Balance
  - Total Income
  - Total Expenses  
- Time-based visualization (Balance Trend - Line Chart)
- Category-based visualization (Spending Breakdown - Doughnut Chart)

---

#### Functionalities:
- Search transactions  
- Filter by type (Income / Expense)  
- Sort by date and amount  
- Add new transactions (Admin only)  
- Handles empty state gracefully  

---

### Role-Based UI
- Viewer → Can only view data  
- Admin → Can add transactions  
- Role switching via dropdown  

---

## 🌙 Enhancements
- Dark mode toggle  
- Local storage persistence  
- Interactive charts using Chart.js  

---

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript  
- Chart.js
